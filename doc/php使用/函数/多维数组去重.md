- 利用array_unique和序列化serialize和反序列化去重

- demo

```
<?php
$a = array(
     array('feed_id' => 123,'text' => '123'),
     array('feed_id' => 123,'text' => '123'),
     array('feed_id' => 234,'text' => '342'),
     array('feed_id' => 234,'text' => '342'),
    );
$unserializeArrs = array_map('unserialize',array_unique(array_map('serialize',$a)));
var_dump($unserializeArrs);
?>
```
